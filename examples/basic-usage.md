### Preferences
```
Remember that I prefer TypeScript over JavaScript
```

### Skills
```
Note that I'm proficient in Python and Node.js
```

## Using Agents

### Automation Agent
```
/agent:automation Check all running services
```

### Researcher Agent
```
/agent:researcher Find best practices for Node.js authentication
```

### Communicator Agent
```
/agent:communicator Draft a response to the project email
```

### Developer Agent
```
/agent:developer Review code in src/utils
```
